# CourtPath

CourtPath is a trust-centered court workflow prototype built to help public users and partner organizations organize a matter, understand next steps, and escalate to real human help when risk rises.

## What is in this repo

- `courtpath-app.html` — clickable product prototype with theme toggle, mobile navigation, modals, and a news-upgrade panel.
- `docs/` — rebuilt PRD, roadmap, founder package, launch checklist, and pitch deck files.
- `pitch/` — investor-facing HTML slides and markdown summary.

## Product direction

The product is designed around an Illinois-first release, matter-centered workflow, official-source grounding, filing readiness, and organization handoff rather than black-box legal advice.

## GitHub publishing

1. Create a new GitHub repository named `courtpath`.
2. Upload the contents of this folder.
3. Set `courtpath-app.html` as the main preview entry, or rename it to `index.html` for GitHub Pages.
4. Enable GitHub Pages from the root branch if you want a public preview.
5. For Vercel or Netlify, deploy as a static site and set the output root to this folder.
