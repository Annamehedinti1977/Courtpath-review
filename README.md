# CourtPath GitHub Pages Package

This folder is ready to upload to a GitHub repository for GitHub Pages hosting.

## Files included
- `index.html` — public review site homepage
- `courtpath-app-prototype.html` — main prototype demo
- `courtpath-master-index.md` — project map
- `courtpath-prd.md` — product requirements
- `courtpath-pitch-deck-draft.md` — pitch narrative
- `courtpath-landing-page-copy.md` — landing-page messaging

## GitHub Pages setup
1. Create a new GitHub repository, for example `courtpath-review`.
2. Upload all files from this folder to the root of that repository.
3. In GitHub, open **Settings** → **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and the `/ (root)` folder, then save.
6. Wait for GitHub Pages to publish the site.
7. Your public URL will look like `https://YOUR-USERNAME.github.io/courtpath-review/`.
